# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial package structure with src layout
- Support for Excel files (.xlsx, .xlsm)
- Support for OpenOffice files (.ods)
- VBA macro detection and analysis
- Formula detection and analysis
- AI-powered security scoring using Claude
- Markdown report generation
- Sanitized copy creation with highlighted suspicious cells
- Command-line interface
- Python API
- Comprehensive test suite with pytest
- Code quality tools (ruff, black, mypy)
- GitHub Actions CI/CD pipeline
- MIT license

## [0.1.0] - 2025-01-XX

### Added
- First public release
- Basic macro and formula detection
- Claude AI-powered analysis
- Report generation
- Spreadsheet sanitization
